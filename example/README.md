# Example App

This folder contains a full React Native example project that consumes the local library package.

## Setup

From repo root:

```bash
npm install
cd example
npm install
```

## Run

```bash
npm run start
npm run ios
npm run android
```

## Notes

- The example depends on the local library via `"emoji-main-color-react-native": "file:.."`.
- Metro is configured to watch the repo root so edits in `src/` are picked up.
